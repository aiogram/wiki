aiogram это современный и полностью асинхронный фреймворк для разработки чат-ботов для Telegram Bot API на языке программирования Python с использованием asyncio.

 

aiogram сейчас имеет 2 активные версии:

v2.8 and v.3.0a4

 

Для получения помощи можно обращаться в официальные чаты сообщества - на английском или русском языках.
