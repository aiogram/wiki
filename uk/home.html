Вітаю на сторінці aiogram

aiogram це сучасний та повністю асинхронний фреймворк для розробки чат-ботів для Telegram Bot API на мові програмування Python з asyncio.

aiogram наразі має 2 активні версії:

v2.8 та v.3.0a4

 

Для отримання допомоги можна звертатись в офіційні чати англійськомовної або російськомовної спільноти.
